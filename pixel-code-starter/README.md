
# Pixel Code Starter

Template base para criação de Landing Pages e Sites.

## Stack

* Next.js
* TypeScript
* Tailwind CSS
* GSAP
* Lenis

## Estrutura

```txt
public/
├── icons/
├── images/
└── videos/

src/
├── app/
├── components/
├── animations/
├── constants/
├── data/
├── hooks/
├── lib/
└── types/
```

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

Abrir:

```txt
http://localhost:3000
```

## Build

```bash
npm run build
```

## Objetivo

Este projeto serve como base para todos os projetos da Pixel Code.

Fluxo:

Starter → Cliente → Desenvolvimento → Deploy

Exemplos:

* Landing Pages
* Sites Institucionais
* Portfólios
* Personal Branding
* Negócios Locais
